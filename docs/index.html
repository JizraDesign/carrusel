Acerca de "Paginador tipo carrusel by Jizra" 0.01.2020.00 :

"Paginador tipo carrusel by Jizra" es un paginador horizontal con el clásico efecto de carrusel infinito construido en Html 5, Css 3 y javascript.
Esta es la primera versión beta de este paginador por tanto cuenta con un funcionamiento muy básico, puede tener fallos y no funcionar en todos los navegadores.

GitHub Descargar
Instrucciones :
1: Agrega la etiqueta de estilos entre tu etiqueta <head>
                
    
<link rel="stylesheet" href="https://jizradesign.github.io/releases/jd-carrusel.min.css">
<script src="https://kit.fontawesome.com/1f8614daef.js" crossorigin="anonymous"></script>
2: Agrega el maquetado html donde quieras poner tu paginador (puedes asignar el nombre que desees al id esto es solo un ejemplo).

    
<section id="cont__paginador>
    <article class="item__paginador">
        
    </article>
</section>
3:Repite este fragmento dentro de la etiqueta <div id="cont__paginador"> por cada elemento que quieras agregar (puedes asignar el nombre que desees a la clase esto es solo un ejemplo, solo recuerda que todos los elementos deben llevar la misma clase).

    
<article class="item__paginador">

</article>
4: Agrega el Script antes del cierre de tu etiqueta </body>
               
<script src="https://jizradesign.github.io/releases/jd-carrusel.min.js"></script>
5: crea un script con la funcion: jdSider('#jd-carrusel','jd-item__carrusel','5000');
               
<script

    window.addEventListener('load', ()=> {

        jdSider('#cont__paginador','item__paginador','5000');

    });

></script>
Como funciona? :
La función jdSider('#cont__paginador','item__paginador','5000'); requiere tres parámetros:

id del elemento contenedor o padre jdSider('#cont__paginador','item__paginador','5000');

class de los elementos hijo jdSider('#cont__paginador','item__paginador','5000');

tiempo en mili segundos para cambiar de elemento('#cont__paginador','item__paginador','5000');
